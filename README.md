# Duelo Pokémon

Mini-game de duelo Pokémon baseado em atributos simples.

Objetivo principal: <br>
-Treinar habilidades de Javascript com if/else, array, funções e objetos. 
<br>
-Estruturação e estilização da página com HTML e CSS.<br>
